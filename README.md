# Cybersecurity Log Monitoring

## Visão Geral
Este projeto simula atividades realizadas por um Analista de Segurança em um Security Operations Center (SOC),
com foco na análise de logs em ambientes Linux. O objetivo é identificar eventos suspeitos, como tentativas de
acesso indevido e possíveis ataques de força bruta, aplicando conceitos de Blue Team e boas práticas de
Segurança da Informação.

## Objetivos
- Analisar logs de autenticação do Linux
- Identificar padrões suspeitos de acesso
- Simular alertas de segurança
- Documentar incidentes e procedimentos
- Aplicar conceitos de SOC e resposta a incidentes

## Tecnologias Utilizadas
- Linux
- Python
- Git e GitHub

## Estrutura do Projeto

cybersecurity-log-monitoring
├── logs/
├── scripts/

├── docs/

## Funcionamento
1. Logs de autenticação são analisados
2. Tentativas suspeitas são identificadas
3. Alertas são gerados
4. Um relatório de incidente é produzido
